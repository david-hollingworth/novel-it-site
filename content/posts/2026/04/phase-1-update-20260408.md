Version v0.3.0 marked the final feature version for phase 1. So what's next?

Testing it what's next. Every requirement has a set of tests written using [Gherkin](https://cucumber.io/docs/gherkin/) syntax. This is a plain text description of the tests that can be automated to run as part of a continuos integration suite. However, there's a lot of work involved in setting up the steps required for the automation, and so a decision was made to execute the test scripts manually.

That's what I'll be doing for the next while, manually executing all the tests. The results will be recorded on the [Test Results](https://david-hollingworth.github.io/novel-it-docs/test-results/) page in the documentation site. Progress can also be viewed on the [Trace Matrix](https://david-hollingworth.github.io/novel-it-docs/traceability/) which links the features to requirements to the test results.
